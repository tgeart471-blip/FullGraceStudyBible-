# Full Grace Study Bible

A comprehensive, AI-powered Bible study application built with React Native and Expo.

## Features

- **AI Insights**: Deep exegesis, word studies, and practical applications powered by Google Gemini.
- **AI Chat Assistant**: Ask biblical questions and get scholarly answers in real-time.
- **Interactive Maps**: Explore the journeys of Jesus and Paul with visual markers.
- **Reformation Insights**: Study Martin Luther's 95 Theses and core theological teachings.
- **Daily Devotionals**: Receive daily inspiration and spiritual reflections.
- **Memorization Tool**: Save and practice verses to hide the Word in your heart.
- **Multiple Translations**: Access KJV, ASV, BBE, WEB, and more.
- **PWA Support**: Install as a Progressive Web App on iOS and Android for instant access.

## Getting Started

### Prerequisites

- Node.js
- Expo CLI (`npm install -g expo-cli`)
- A Google Gemini API Key

### Installation

1. Clone the repository.
2. Navigate to the project folder: `cd full-grace-project`
3. Install dependencies: `npm install`
4. Create a `.env` file and add your API keys:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   YVP_APP_KEY=your_youversion_app_key_here (optional)
   ```
5. Start the application: `npm start`
